1. Open docker desktop
2. Run these commands to download the image and enter the node container with npm
> docker pull node:22-alpine
3. Run these commands in the container
> npx create-react-app frontend
> cd frontend
> npm install
> npm start

> docker run -it --rm -v "${PWD}:/app" -w /app -p 3000:3000 node:22-alpine sh -c "cd frontend && npm start -- --host 0.0.0.0"


4. To test backend is working (using image and style to get result.png)
> curl.exe -X POST -F "image=@C:/Users/prana/Downloads/Dada.jpeg" -F "style=style2.jpg" http://127.0.0.1:5000/api/transfer --output result.png
